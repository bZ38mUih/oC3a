Семейство операционных систем Windows - это результат интеллектуального труда огромного числа программистов, инженеров, дизайнеров корпорации Microsoft. За время существования Windows прошла большой путь в конкурентной борьбе за пользователей, постоянно улучшалось качество и интерфес пользователя OS, совершенствовались приложения, входящие в ее состав. Именно благодаря такому подходу так велика популярность Windows в мире, на настольных компьютерах более 80%.

Windows OS обычно устанавливают на настольные компьютеры и ноутбуки. В состав Windows уже включены браузер, медиа-проигрыватель, приложение Outlook для работы с электронной почтой и др. Множество как платных, так и бесплатных программ для Windows дают пользователям возможность использовать OS для решения практически любых повседневных и специальных задач с помощью компьютера.

Какую Windows выбрать

Версию Windows OS следует выбирать исходя из задач, которые будут решаться на вашем ПК и системных требований к компьютеру (желаемое и возможное). Учитывая тот факт, что устаревшая версия Windows XP больше не поддерживается компанией Microsoft, которая задает тренд на установку современных версий своей системы, следуя за ней разработчики будут выпускать свои программы для современных ОС, следует устанавливать Windows 10. 10-ка быстрее стартует, в нее возвращено меню Пуск и другие приятные опции. Может показаться что сборка Windows 10 будет менее полной и сырой и из-за этого компьютер будет глючить, но не переживайте, это все можно отнести к предыдущим версиям windows-7 и 8.

32-bit или 64-bit

Если на вашем компьютере установлено 8 ГБ и более оперативной памяти, то следует выбирать 64-битную версию Windows. 32-битные операционные системы не поддерживают более 4 ГБ ОЗУ. Для современных игр с качественной графикой или обработки в большом количестве видео высокого разрешения следует устанавливать windows-10 x64

Для интернет-серфинга, просмотра видео, работы с документами, даже для разработки web-, net- приложений может сгодиться простой ПК с двух ядерным процессором (например intel core i3 2350M и 2 ГБ ОЗУ). Следует выбирать Windows-10 x32

Если вы собираетесь использовать старенькие монитор, принтер, сканер или другую технику, лучше убедитесь что они будут поддерживаться 64-битной ОС и вы сможете найти для них подходящий драйвер.

Home или Pro (Enterprice)

В версиях Pro обычно используется доменная учетная запись, рабочий компьютер подключается к локальный сети и имеет доступ к ресурсам этой сети в соответствии со своими разрешениями и групповыми политиками. Такая сеть управляется c единственного компьютера, называемого сервером. Для некорпоративных пользователей преимущества установки Pro неочивидно, но при возможности все-таки лучше устанавливать максимальную версию Pro (Enterprice)

Специальные версии Windows Server используются на компьютерах в качестве серверов сети, их нет необходимости устанавливать на компьютеры обычных локальных и корпоративных пользователей, даже если компьютер используется для разработки и на него (на локальный хост) устанавливаются сервера.

Как установить Windows

  • Убедитесь что конфигурация вашего компьютера соотвествует системным требования ОС.
  • Для начала вы скачиваете образ ISO диска с операционной системой (откуда скачать см. выше).
  • Устанавливать Windows лучше с флэшки, для записи образа на usb воспользуйтесь утилитой windows usb dvd download tool
  • При необходимости установите в BIOS соответствующий режим загрузки. Обычно это раздел boot, опции Configuration и Priority
  • Процесс установки интуитивно понятен и практически не требует участия человека, просто следуйте инструкциям, что дает вам установщик
  • Активируйте Windows свои ключом активации, купите ключ или воспользуйтесь активатором Windows

Трудности при установке/настройке Windows

Обычно установка Windows проходит без каких-либо проблем и занимает не более часа, но затем часто приходится обновлять драйвера. Если ваш компьютер подключен к интернету, можно воспользоваться автоматическим поиском драйверов из диспетчера задач, обычно таким путем приходится искать драйвера для видео-карты. Часто возникают проблемы с разрешением монитора, использовании двух мониторов, когда стандартный графический драйвер Microsoft не оптимален. Бывают проблемы со старой техикой при переходе от 32-битной к 64-битной версиям.
