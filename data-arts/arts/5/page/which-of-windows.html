Семейство операционных систем windows - это результат интеллектуального труда огромного числа программистов, инженеров, дизайнеров корпорации Microsoft. За время существования Windows прошла большой путь в конкурентной борьбе за пользователей, постоянно улучшалось качество и интерфес пользователя OS, совершенствовались приложения, входящие в ее состав. Именно благодаря такому подходу так велика популярность windows в мире, на настольных компьютерах более 80%.

Windows OS обычно устанавливают на настольные компьютеры и ноутбуки. В состав windows уже включены браузер, медиа-проигрыватель, приложение outlook для работы с электронной почтой и др. Множество как платных так и бесплатных программ для windows дают пользователям возможность использовать OS для решения практически любых повседневных и специальных задач с помощью компьютера.

Какую windows выбрать

Версию Windows OS следует выбирать исходя из задач, которые будут решаться на вашем ПК и системных требований к компьютеру (желаемое и возможное). Учитывая тот факт, что устаревшая версия windows XP больше не поддерживается компанией Microsoft, которая задает тренд на установку современных версий своей системы, следуя за ней разработчки будут выпускать свои программы для современных OS, следут устанавливать Windows 10. 10-ка бастрее стартует, в нее возвращено меню Пуск и другие приятные опции. Может показаться что сборка windows 10 будет менее полной и сырой и из-за этого компьютер будет глючить, но не переживайте, это все можно отнести к педыдущим версиям windows-7 и 8.

32-bit или 64-bit

Если на вашем компьютере установлено 8 ГБ и более оперативной памяти, то следует выбирать 64-битную версию Windows. 32-битные операционные системы не поддерживают более 4 ГБ ОЗУ. Для современных игр с качественной графикой или обработки в большом количестве видео высокого разрешения следует устанавливать windows-10 x64

Для интернет-серфинга, просмотра видео, работы с документами, даже для разработки web-, net- приложений может сгодиться простой ПК с двухядерным процессором (например intel core i3 2350M и 2 ГБ ОЗУ). Следует выбирать Windows-10 x32

Если вы собираетесь использовать старенькие монитор, принтер, сканер или другую технику, лучше убедитесь что они будут поддерживаться 64-битной ОС и вы сможете найти для них подходящий драйвер.

Home или Pro (Enterprice)

В версиях Pro обычно используется доменная учетная запись, рабочий копьютер подключается к локальный сети и имеет доступ к ресурсам этой сети в соответствии со своими разрешениями и групповыми политиками. Такая сеть управляется c единственного компьютера, называемого сервером. Для некорпоративных пользователей преимущества установки Pro неочивидны, но при возможности все-таки лучше устанавливать максимальную версию Pro (Enterprice)

Специальные версии Windows Server используюся на компьютерах в качестве серверов сети, их нет необходимости устанавливать на копьютеры обычных локальных и корпоративных пользователей, даже если копьютер используется для разработки и на него (на локальный хост) устанавливаются сервера.

Вы можете купить лицензию, но рядовой российский пользователь вряд ли захочет платить приличные деньги, за то, что можно получить бесплатно. Как установить Windows

  • Убедитесь что конфигурация вашего комьютера соотвествует системным требования ОС.
  • Для начала вы скачиваете образ ISO диска с операционной системой (откуда скачать см. выше).
  • Устанавливать Windows лучше с флэки, для записи образа на usb воспользуйтесь утилитой windows usb dvd download tool
  • При необходимости установите в BIOS соответствующий режим загрузки
  • Процесс установки интуитивно понятен и практически не требует участия человека, просто следуйте инструкциям, что дает вам установщик
  • Активируйте Windows свои ключем активации, купите ключ или воспользуйтесь активатором Windows

Трудности при установке/настройке Window

Обычно установка Windows проходит без каких-либо проблем и занимает не более часа, но затем часто приходится обновлять драйвера. Если ваш комьютер подключен к интернету, можно воспользоваться автоматическим поиском драйверов из диспетчера задач, обычно таким путем приходится искать драйвера для видео-карты. Часто возникают проблемы с рарешением монитора, использовании двух мониторов, когда стандартный графический драйвер Microsof не оптимален. Бывают проблемы со старой техикой при переходе от 32-бинтной к 64-битной версиям.
