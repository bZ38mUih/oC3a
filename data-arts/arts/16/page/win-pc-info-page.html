Клиентская утилита

win-pc-info.exe позволяет собирать различную информация о компьютерах на Windows OS, такую как:

  • Переменные окружения
  • Сведения о системе
  • Сведения об аппаратуре
  • Список процессов
  • Список служб

Скриншот программы

Собранные данные можно сохранить в .json диаг-файл для последующей загрузки в программу или на сервер для просмтра и сравнения.

Программа не требует установки и может быть запущенна с флэшки. В папке с exe-файлом программы должен находиться Newtonsoft.Json.dll (библиотека для работы с .json-файлами)

Серверное приложение

Авторизированные пользователи могут загрузить .json диаг-файл на сервер для получения дополнительных сведений о ПК и их построчного сравнения в соответствии с выбранными опциями.

Перспективы развития проекта

  • Расчет стоимости ПК: в приложение можно легко добавить данные о стоимости аппаратуры и расчитать общую стоимость компонетов компьютера
  • Лучший ПК: в приложение легко можно добавить о данные производительности аппаратуры на основе публикуемых в интернет тестов, сравнивать и выбирать лучший ПК
  • Углубленный анализ: при накоплении достаточного количества сведений из диаг-файлов возможно разработка алгоритмов для углубленного автоматического анализа системы и компонентов для поиска и устранения неисправностей
