Командная строка Windows - это специальная программа включенная в состав ОС windows, предназначенная для управления компьютером путем текстовового ввода команд. Интерфейс командной строки - это консоль. Переимущества такого подхода - низкое потребление системных ресурсов.

Для запуска конадной строки можно испоьзовать сочетание клавиш Win + R, в окне выполнить необходимо набрать cmd

Несмотря на то, что существует большое количество команд для управления практически любыми настройками операционной системы, на смену командной строки приходит более современная технология power shell.

Список команд:

Общие

[+]
Cls
Очищает содержимое экрана.
findstr
Поиск строк в файлах. В качестве фильтра используется регулярное выражение.
показать[+]
tasklist | findstr google
>
Сохраняет вывод в текстовый файл.
показать[+]
tasklist > c:\tasklist.txt wmic /? > c:\wmic_help.txt
/?
Справка о команде.
показать[+]
tasklist /?
Exit
Завершает программу CMD.EXE (интерпретатор команд) или текущий пакетный файл-сценарий.

Файлы и папки

[+]
Cd
Выводит имя или изменяет текущий каталог.
Copy
Копирование одного или нескольких файлов в другое место.
показать[+]
copy c:\test\111.txt c:\test\222.txt
Del
Удаление одного или нескольких файлов.
Dir
Вывод списка файлов и подкаталогов в указанном каталоге.
Chdir
Выводит имя или изменяет текущий каталог.
Attrib
Отображает или изменяет атрибуты фалов.
Mkdir
Создание каталога.
Move
Перемещение файлов и переименование файлов и папок
Rmdir
Удаление каталога.
Rename
Переименование одного или нескольких файлов.
Mklink
Создает символьную ссылку.
Fc
Сравнение двух файлов.

Диски, файловые системы, разделы дисков

[+]
Label
Создание, изменение и удаление меток тома для дисков.
Defrag
Служит для оптимизации и объединения фрагментированных файлов на локальных томах в целях повышения производительности системы.
Vol
Вывод метки и серийного номера тома для диска.
Format
Форматирование диска для работы с Windows.
Recover
Восстановление сохранившихся данных на поврежденном диске.
Bcdboot
средство создания и восстановления файлов данных конфигурации загрузки.
Bcdedit
изменяет хранилище данных конфигурации загрузки.
Bootsect
Средство восстановления загрузочного сектора
Diskpart
Утилита для работы с дисковыми данными

Время, временные зоны, переменные окружения

[+]
Set
Вывод, задание и удаление переменных среды.
Date
Вывод или изменение даты.
Time
Вывод или изменение времени.
Tzutil
Программа Windows Time Zone

Процессы и службы

[+]
Tasklist
Отображает список процессов, которые сейчас выполняются на локальном или удаленном компьютере.
показать[+]
tasklist /m tasklist /v /fo csv
Tskill
Прекращение процесса.
Qprocess
Отображает информацию о процессах.
Sc
SC представляет собой программу командной строки, используемую для связи со службами и диспетчером управления службами.
Start
Запускает указанную программу или команду в отдельном окне.

Интерфейсы и протоколы

[+]
Ipconfig
Отображает информацию по каждому адаптеру TCP/IP.
arp
Отображение и изменение таблиц преобразования IP-адресов в физические, используемые протоколом разрешения адресов (ARP).
показать[+]
arp -a arp -a 100.104.12.1
Ping
Посылает ICMP эхо-запрос удаленному хосту для проверки IP-уровня.
Tracert
Проверка маршрута ICMP пакета путем увеличения времени жизни пакета (TTL).
показать[+]
tracert -d -h 7 10.45.176.2
Netstat
Отображение статистики протокола и текущих сетевых подключений TCP/IP.
показать[+]
netstat -aon
Route
Обработка таблиц сетевых маршрутов.
показать[+]
route print
Getmac
Эта команда позволяет администратору отобразить MAC-адреса сетевых адаптеров компьютера.
Nslookup
Позволяет опрашивать произвольно указываемые сервера.
показать[+]
nslookup mail.ru

Конфигурация системы и железа

[+]
Systeminfo
Отображает сведения о конфигурации операционной системы локальный или удаленный компьютер, включая уровни пакета обновления.
wmic
Windows Management Instrumentation Command line. Отображение и изменение параметров системы и аппаратуры.
показать[+]
wmic memorychip list wmic path win32_VideoController get name wmic computersystem list wmic nic where NetEnabled=true get Name, Speed
mem
Показывает количество использованной и свободной памяти.
Schtasks
Позволяет администратору создавать, удалять, изменять и опрашивать запланированные задачи в локальной или удаленной системе.
Driverquery
Эта команда позволяет администратору просмотреть список установленных драйверов устройств.
Licensingdiag
Инструмент для создания лог-файлов содержащих информацию о лицензии и активации Windows.
Reg
Используется для работы с реестром Windows. Позволяет добавлять, изменять, экспортировать реестр.
Shutdown
Задание параметров выхода из системы и перезагрузки компьютера.

Пользователи и группы

[+]
Gpresult
Отображает результирующую политику (RSoP) для указанного пользователя и компьютера.
Quser
Отображение информации о пользователях, вошедших в систему.
Qwinsta
The qwinsta command is used to display information about open Remote Desktop Sessions. The qwinsta command is available in Windows 8, Windows 7, Windows Vista, and Windows XP.
